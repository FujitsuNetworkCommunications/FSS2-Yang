Getting started
---
This repository contains FSS2 Yang models that highlight the capabilities of Fujitsu's 1FINITY Network Elements. 

The repository is structured as follows:
```
|-- FSS2-API-Yang
|   |-- Product
|   |   -- Product_version
|   |       -- yang
|   |           |-- docs
|   |           |-- standards
|   |           |-- third_party
```
